# NovaMarket

Aplicación Flask lista para desplegarse gratuitamente en Render. Usa SQLite y carga automáticamente comercios, productos y promociones de demostración sin datos personales.

## Cuentas de demostración

- Cliente: `cliente@demo.local` / `demo123`
- Vendedor: `vendedor@demo.local` / `demo123`
- Administrador: `admin@demo.local` / `demo123`

En Render gratuito, los cambios en SQLite pueden reiniciarse cuando el servicio se vuelve a desplegar.
